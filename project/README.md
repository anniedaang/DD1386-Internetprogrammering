# Projekt

[Projektet](https://canvas.kth.se/courses/36975/assignments/229578/) ska lämnas in här före presentationen. Båda studenterna ska kunna redogöra för programmets alla delar. De formella kraven finns på Canvassidan ovan. Utgå från strukturen i Lab4 och tänk på att hålla en någorlunda klar githistorik, där det tydligt framgår vilka som har arbetat på projektet och vilken insats de har gjort. Försök även att göra nya revisioner ofta och linta koden alltid dessförinnan. Lintningsreglerna inkl. all nödvändig konfiguration ska hämtas från Lab4-repot. Avslutningsvis, glöm inte att uppdatera [README.md](./README.md)-filen med en kort beskrivning av projektet och tveka inte att be om hjälp om det behövs. _Lycka till!_
